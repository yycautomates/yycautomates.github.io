# yycautomates.github.io

This is the repository holding the source code for static site yycautomates.ca, hosted with GitHub pages. It makes use of Hugo, and the [universal hugo theme](https://github.com/devcows/hugo-universal-theme).

## Process to clone locally and add/change content

```
$ git clone git@github.com:yycautomates/yycautomates.github.io.git

$ cd yycautomates.github.io

$ <update/copy existing content to use as a template for new content>

View updated content:

$ hugo server

Navigate to localhost:1313
Test/Verify

$ git add <content>
$ git commit -m "comment"
$ git push origin main
```
